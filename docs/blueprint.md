# **App Name**: St. Lucia Elections Hub

## Core Features:

- Candidate Profiles & Party Info: Displays candidate profiles with background, party affiliation, and policy positions. Integrated admin panel to manually input/update the information.
- Election Timeline: Countdown timer to the 2026 election day and important deadlines (registration, absentee voting, etc.).
- Polling Data Display: Visual representation of poll results, including historical trends. Integrated admin panel to manually input/update polling data.
- Election Prediction Tool: Generative AI-powered predictions based on current polls and historical data. The AI tool determines the impact of recent news or events to help predict the outcome.
- Past Election Results: Displays comprehensive results of all St. Lucian General Elections since 1974, broken down overall and by constituency.
- Voting District Information: Information on each voting district/constituency including maps, demographics, and polling locations. Integrated admin panel to manually input/update the information.
- Admin Panel: Backend for manual input and management of party, candidate, election, polling data, constituency information, and election results.

## Style Guidelines:

- Primary color: Deep Ocean Blue (#2980B9) to evoke trust, stability, and reliability.
- Background color: Very light desaturated blue (#EBF5FB), nearly white, providing a clean, neutral backdrop.
- Accent color: Sun Yellow (#F1C40F) to highlight key information and call to action buttons.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and techy feel.
- Body font: 'Inter' (sans-serif) to ensure readability for longer text sections. Use with Space Grotesk.
- Use clean and modern icons representing voting, candidates, and constituencies. Consistent style with a focus on clarity.
- Clean, card-based layout to present information in an easily digestible manner. Responsive design for access across devices.
